# Greekfolio - Creative Studio Portfolio

## 🎯 Overview

Greekfolio is a professional portfolio website designed to showcase creative studio services, including digital art, design, and development solutions. Built with clean, semantic HTML and modern CSS, it delivers an engaging user experience with a sleek dark theme and interactive elements. A modern, responsive web portfolio for a creative studio specializing in UI/UX design, development, and strategic creative solutions.

## 🚀 Getting Started

### Prerequisites

No installation required! This is a static HTML/CSS website.

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ahammad204/new-web.git
   cd new-web
   ```

2. **Open in browser:**
   - Double-click `index.html`, or
   - Right-click and select "Open with Live Server" (VS Code), or
   - Drag `index.html` into your browser

### Local Development

For development with live reload, use VS Code extensions:
- **Live Server** - Quick start local development server
- **Live Sass Compiler** - For CSS preprocessing (if expanding to SCSS)



To deploy your own version to GitHub Pages:

1. Push code to GitHub repository
2. Go to repository Settings → Pages
3. Under "Build and deployment", select:
   - Source: `Deploy from a branch`
   - Branch: `main` (or your default branch) with folder `/ (root)`
4. Save - your site will be live at `https://<username>.github.io/new-web/`

### Alternative Hosting Options

- **Netlify** - Drag and drop deployment (free tier available)
- **Vercel** - Git-connected deployment
- **Firebase Hosting** - Google's cloud hosting solution

## 🎨 Customization Guide

### Colors & Theming

Edit `style.css` to customize colors:

```css
/* Primary Dark Background */
.navbar { background-color: #0D0A0C; }

/* Accent Color (Purple) */
.btn { background-color: blueviolet; }

/* Text Colors */
body { color: azure; }
```

### Typography

Update font sizes and weights in `style.css`:

```css
.titler { font-size: 54px; }  /* Main title */
.menu { font-size: 18px; }    /* Navigation */
```

### Content Updates

1. Replace images in `images/` and `icon/` folders
2. Update text in corresponding HTML sections
3. Modify links in navigation and buttons
4. Update logo in `logo/` folder
